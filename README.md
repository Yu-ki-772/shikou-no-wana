## アプリURL

https://shikou-no-wana.vercel.app



## アプリ開発の背景
これまでの人生の中で、 **「客観的かつ冷静に考え直してみたら、こっちの方が適切な選択だった」** という経験をしたことがある人は多いのではないでしょうか。その選択には、ご自身が気付かぬうちに持っている、 **「思考の偏り」** が関係している場合があるはずです。

「思考の偏り」を気軽に楽しみながら考え直すきっかけをつくりたいと考え、本アプリを開発しています。

## クリアしたい課題
**「なんとなく正しいと思って選んだ答えが、実は偏っていた」** という体験を、誰でも気軽にできるようにすること。

## 工夫した点
① 「思考の偏り」は、やや固いテーマのように感じられそうだという懸念があったため、クイズ形式にすることで直感的に楽しみながら体験できるようにしました。

② また、画面のデザインやアニメーションに関しては、推理ゲームのような雰囲気を出し、楽しみやすくなるように意識しました。

## 技術スタック

- Next.js
- TypeScript
- React
- TailwindCSS
- Docker（開発環境の構築）
- Vercel（本番環境）

## 技術選定の理由
### ■React
コンポーネント設計・状態管理等の豊富なエコシステムを持っているため。<br>
本アプリでも **useStateによる画面遷移の管理** や **useEffectによるアニメーション制御** を実装しています。
### ■Next.js
React公式が推奨するフレームワークであり、業界での採用実績が高いため。

### ■TypeScript
型安全性を確保し、予期せぬバグを防ぐため。

### ■Vercel
Next.jsとの親和性が高く、設定不要で自動デプロイが可能なため。
